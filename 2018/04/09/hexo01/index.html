hexo生成博文插入图片

2018-04-09
  • hexo

原由

因为发现上传的博客里面的图片居然显示不来，试着搜索了下本地方式引用。发现了hexo-asset-image的插件来加载本地图片。

步骤

  • 把主页配置文件_config.yml 里的post_asset_folder:这个选项设置为true;

  • 在你的hexo目录下执行：npm install hexo-asset-image –save，这是下载安装一个可以上传本地图片的插件；

  • 等待一小段时间后，再运行hexo n “aaa”来生成md博文时，/source/_posts文件夹内除了aaa.md文件还有一个同名的文件夹;

  • 最后在aaa.md中想引入图片时，先把图片复制到aaa这个文件夹中，aaa.md中按照markdown的格式引入图片：![你想输入的替代文字](aaa/图片名.jpg)；

  • 注意： aaa是这个md文件的名字，也是同名文件夹的名字。只需要有文件夹名字即可，不需要有什么绝对路径。你想引入的图片就只需要放入aaa这个文件夹内就好了，很像引用相对路径。

  • 最后检查一下，hexo g生成页面后，进入public里的.html文件中查看相关字段，可以发现，html标签内的语句是<img src="2018/04/09/aaa/图片名.jpg">，而不是<img src="aaa/图片名.jpg>。这很重要，关乎你的网页是否可以真正加载你想插入的图片。
