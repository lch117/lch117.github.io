Element-Ul

饿了么前端团队推出的一款基于Vue.js 2.0 的桌面端UI框架，手机端有对应框架是Mint UI ，Star 数将近 33K；

Mint UI

由饿了么前端团队推出的 Mint UI 是一个基于 Vue.js 的移动端组件库，提供了用于构建移动应用程序的 CSS 和 JS 组件。当全部导入时，压缩后的代码只有月 30KB（JS+CSS），支持单个组件的导入。Star 数将近 13K。
  • 文档地址：https://mint-ui.github.io/#!/zh-cn
  • github地址：https://github.com/ElemeFE/mint-ui/

iView

iView 是一套基于 Vue.js 的开源 UI 组件库，主要服务于 PC 界面的中后台产品， Star 数将近 19K。
特性：高质量、功能丰富，友好的API、自由灵活地使用空间，细致、漂亮的UI，事无巨细的文档，可自定义主题。
谁在使用：TalkingData、阿里巴巴、腾讯、京东、滴滴出行、美团、新浪、联想。
  • 文档地址：https://www.iviewui.com/docs/guide/install

Vue Material

Vue Material 是一个轻量级的框架， 建立在谷歌的 Material Design 基础上。 设计强大的和美观的web应用并适用于不同的屏幕，Star 数为7K多。

  • 文档地址：http://vuematerial.materializecss.cn/#/

Vuetify

使用一个Vuetify Vue CLI安装包 (基于官方示例) 立即开始您的项目。 Vuetify 支持SSR（服务端渲染）,SPA（单页应用程序）,PWA（渐进式Web应用程序）和标准HTML页面，Star 数将近 15K。

vux
