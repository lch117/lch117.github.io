vue2.0中如何使用swiper插件

2019-01-14
  • vue

安装

  • npm install Swiper –save

引用组件

  • import Swiper from ‘swiper’;
  • 或者直接引用方法：import Swiper from ‘../static/js/swiper.min.js’（使用这种方法就可以省去安装）

引用样式文件

  • 在需要的轮播的vue文件里引用swiper.min.css，例如：@import'../assets/css/swiper.min.css';

遇到的问题

  • 使用swiper做轮播图时，异步调用数据时导致图片不能滑动;
  • 不能滑动的原因：因为swiper提前初始化了，那个时候还没有数据，当数据调出来时已经没用了。

解决方法

  • 在数据调用结束后再进行swiper初始化；
  • 还要用到$nextTick,它会在数据变化以后，DOM更新以后进行回调函数，不然的话轮播还是会出现错乱的情况。
